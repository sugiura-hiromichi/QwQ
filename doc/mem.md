# 入寮案内を読んだ感想

入寮案内を読んでまず最初に、楽しそうだと思った。
話し合いの原則や様々な部やクラブ？など（自分は吉田寮技術部に興味がある）本当に自分たちで自分たちの生活を作っているんだなと感じた。
敬語の非推奨については、経験(吉田寮にどのような人たちがいるのかというのも含めて)がないのであまり想像がつかないが、自分に合っていると感じた。
というのも、自分は小さい頃から学校で、1年、2年の出生の違いで何故このような上下関係が導かれるのだろうかと不思議に思っていた。
特に後輩と接する時どう接すれば良いか少なからず考え、'先輩らしい態度'というものに違和感を感じていた。
話し合いの原則についても興味がある。興味があるというか、自分の肌で感じておきたいという思いがある。(理由)。
…
敬語の非推奨にも通ずるが、吉田寮は何より対話を重視していると感じた。自治寮としての性格が強く出ていると思う。

# 自治についての自分の考え

自分は寮の自治というのは大学生にとって当然の権利だと考えている。寮は寮生にとって自分たちが住む場所な訳で、まだ社会に出ていないとはいえ皆すでに成人している。
稼ごうと思えば自力でで稼ぐ能力があり、話し合いをするのに思慮が足りないという訳も全く無い。
また、自治を通じたコミュニケーションや経験は価値のあるものたと考えている。
何故なら、同じ屋根の下で住む者同士という距離感では相互理解が求められ、その上で意思疎通をする必要がある。
これは、社会生活をする上で広く求められるもので、同時に多くの人が実行できていない難しいことだから。これについて、社会人と学生に違いはないと思う。
こういった、大学の寮に住んでいる学生という立ち位置において、自分たちの住む場所は自分たちが管理するという主張は至極真っ当なもので、認められるべきだと思う。
ここでの管理とは大学、もしくは寮生にとって都合の良い物のことではなく、自治の文字通り、自立して治めることを指している。
だから大学も含め、寮の自治に直接関わらない組織や人間がこの権利を侵害してはいけないと考えていて、それが脅かされている現状を危うく感じる。
