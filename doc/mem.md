# 入寮案内を読んだ感想

入寮案内を読んでまず最初に、楽しそうだと思った。
話し合いの原則や様々な部やクラブ？など（自分は吉田寮技術部に興味がある）本当の自治寮の生活はこうなっているのかと思った。
敬語の非推奨については、経験(吉田寮にどのような人たちがいるのかというのも含めて)
がないのであまり想像がつかないが、自分にとってはありがたい文化だと思った。
というのも、自分は人と会話するのが苦手で、何かあったらすぐに相手の意見に合わせてしまう。
敬語非推奨の文化は、単に寮生同士仲良くしようというものではなく(もちろんそれも大事だけど…)、
どちらかというとより実り多い、より深い、分け隔てない話し合いをするための文化だと捉えている。
これくらい話し合いを重要視する吉田寮だからこそ、こんな自分でも寮生の一員であるという実感を持ち自治にもちゃんと参加できるのではないかと考えている。
まとめると、吉田寮は何より対話を重視していると感じた。

# 自治についての自分の考え

自分は寮の自治というのは大学生にとって当然の権利だと考えている。
寮は寮生にとって正しく自分たちが住む場所な訳で、まだ社会に出ていないとはいえ皆すでに成人している。
稼ごうと思えば自力でで稼ぐ能力があり、話し合いをするのに思慮が足りないという訳も全く無い。
つまり、自治をするにあたって自分たちでちゃんと社会的責任を背負うことが十分に出来ると考えられる。
また、自治を通じたコミュニケーションや経験は価値のあるものたと考えている。
何故なら、同じ屋根の下で住む者同士という距離感では相互理解が求められ、その上で意思疎通をする必要がある。
これは、社会生活をする上で広く求められるもので、同時に多くの人が実行できていない難しいことだから。
この’必要性’と’難しさ’という点について、社会人と学生に違いはないと思う。
こういった、大学の寮に住んでいる学生という立ち位置において、自分たちの住む場所は自分たちが管理するという主張は至極真っ当なもので、認められるべきだと思う。

だから大学も含め、寮の自治に直接関わらない組織や人間がこの権利を侵害してはいけないと考えていて、それが脅かされている現状を危うく感じる。
